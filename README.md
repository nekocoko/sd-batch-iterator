# sd-batch-iterator
Stable Diffusion WebUI를 위한 프롬프트 일괄 실행 및 자동화 확장 프로그램입니다. JSON 파일에 정의된 프롬프트 목록을 순차적으로 실행하여 생성을 자동화합니다. / This is a batch prompt execution and automation extension for Stable Diffusion WebUI. It automates generation by sequentially executing a list of prompts defined in a JSON file.
